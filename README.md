# Quote Generator

A React quote generator for browsing Zoho CRM products, building a cart, creating quotes, and viewing existing quote details.

## Prerequisites

- Node.js 20 or newer
- npm

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```
